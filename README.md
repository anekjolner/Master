# Master
Data-driven models for optimization of the PSA process



## Datasets

All .mat-filene is in`data/` and contains input/output-data used to train, validate and test the data-driven models

- `Input_train.mat`, `Input_valid.mat`, `Input_test.mat`: Inputs
- `Output_train.mat`, `Output_valid.mat`, `Output_test.mat`: outputs
- `*_global.mat`: Hole dataset
